# 💰 SmartSpend - Expense Tracker App

**Track your money. Build your future.**

SmartSpend is a modern expense tracking application built using **Compose Multiplatform (CMP)**. It allows users to manage income, expenses, and savings with a clean UI and powerful analytics across multiple platforms.
